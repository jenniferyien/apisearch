Welcome to Searchster!

Searchster allows you to search based on the themes(e.g. movies/shows, music/artists) and displays specific information about them. All information provided comes from various APIs that were available on the web.

Please Enjoy! Click on the theme you would like to search for to get started.

APIs used Searchster

Movies/Shows: OMDb API

Music/Artists: iTunes Search API
